# Changelog

All notable user-facing changes to PaperMonster are documented here.

## 1.0.0 - 2026-07-19

### Added

- Interactive cartoon printer with ten randomized failure outcomes
- Guaranteed correct print and celebration on attempt ten
- Animated paper path, rollers, status lights, jams, shaking, throwing, and retraction
- Locally generated printer sounds with a persistent in-session sound toggle
- Attempt counter, progress meter, incident history, and interrupt-safe reset
- Responsive layout, keyboard-accessible controls, live status updates, and reduced-motion support
- Restrictive static Content Security Policy and dependency-free verification gate
